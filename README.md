# Leet Code Problems
Contains my solutions to Leet Code problems

# The goal
The goal is to provide as many solutions in major programming languages. 
Each folder will have the problem description and solutions in various languages. Please provide references to where you got the solution on the internet if you're not the original author of the code.
The solutions I provide will be the naive implementations, therefore if you plan on implementing optimized solutions, please provide an explanation to your optimizations.
