# Longest Palindromic Substring
Given a string `s`, find the longest palindromic substring in `s`.

**Example 1:**
```
Input: "babad"
Output: "bab"
Node: "aba" is also a valid answer.
```

**Example 2:**
```
Input: "cbbd"
Output: "bb"
```

**Example 3:**
```
Input: s = "a"
Output: "a"
```

**Example 4:**
```
Input: s = "ac"
Output: "a"
```

**Constraints:**
* `1 <= s.length <= 1000`
* `s` consist of only digits and English letters (lower-case and/or upper-case)
